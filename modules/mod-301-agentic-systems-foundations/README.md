# mod-301-agentic-systems-foundations — Solutions

Reference solutions for the Agentic Systems Foundations module (mod-301). These are
design and architecture exercises, so each solution is a worked **model artifact** —
a filled decision matrix, an annotated pattern catalog, a capability map with ADRs,
or a published-system teardown — rather than runnable code.

## Index

- [exercise-01-workflows-vs-agents-decision-matrix](exercise-01-workflows-vs-agents-decision-matrix/README.md)
  — a reusable, frozen-weight decision instrument scored across five candidates plus
  an adversarial trap.
- [exercise-02-orchestration-pattern-catalog](exercise-02-orchestration-pattern-catalog/README.md)
  — annotated entries for the five orchestration patterns, ten task mappings, and
  two composed systems with the workflow/agent boundary marked.
- [exercise-03-tool-vs-subagent-decomposition](exercise-03-tool-vs-subagent-decomposition/README.md)
  — a twelve-capability map for a support-automation platform, a reference
  architecture, three ADRs, and a corrected anti-pattern.
- [exercise-04-reference-architecture-teardown](exercise-04-reference-architecture-teardown/README.md)
  — a teardown of Anthropic's multi-agent research system: patterns, boundary map,
  start-simple critique, and a defensible alternative.

## Using these solutions

Each exercise README follows the same shape: **Approach** (the reasoning and design
choices), **Reference solution** (the worked artifact), **Meeting the acceptance
criteria** (mapped to the learning exercise), **Common pitfalls**, and
**Verification** (how to check a submission, including the `NOTES.md` reflection
prompts). Read the learning module's exercise brief first, attempt the artifact, then
compare against the reference — the value is in the judgment, not in matching the
exact scores or wording.
